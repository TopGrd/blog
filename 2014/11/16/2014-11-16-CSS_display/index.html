#浏览器是如何判断元素是否匹配某个 CSS 选择器?

比如#divBox p span.red{color:red;}，浏览器的查找顺序如下：

  1. 先查找html中所有class=’red’的span元素
  2. 再查找其父辈元素中是否有p元素
  3. 判断p的父元素中是否有id为divBox的div元素，如果都存在则匹配上。
