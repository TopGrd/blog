Tracking.js Javascript的计算机视觉库

Dec 4, 2017

惭愧惭愧，距离上次写博客已经很久很久了。

简介

tracking.js库将不同的计算机视觉算法和技术带入浏览器环境。通过使用现代HTML5规范，让你能够进行实时的颜色跟踪，人脸检测等等。所有这些都使用轻量级核心（〜7 KB）和直观的界面。

快速使用

首先，从官方给的链接里将trackingjs库下载下来,引入到你的html文件中。如果需要进行人脸识别的话，还需引入下载源文件夹下build/data下的几个文件，如下图所示。

当然，现代化前端开发是可能需要使用ES6的import方式引入的，这样需要改下源码，因为tracking.js目前还不支持模块化。首先通过npm install tracking安装，然后从node_modules中找到名为tracking的包，修改build下tracking.js文件，在最后加一行加上: module.exports = window.tracking。这样你就可以在你的js文件中import tracking from 'tracking'了😁。不过注意，文章下面的示例都是使用html引入为例的。
